O Git Bash é um terminal extendido para otimizar o uso do Git.


***Comandos para subir diretórios no GitHub:***
git status           = verificar os status dos diretórios, se possuem alterações;
git add *            = (pode ser . ou * ou -a) adiciona os diretórios para o controle de versão local;
git commit -m        = (-m é para add mensagem) aponta para a tree, parente, autor, mensagem, timestamp;
git push origin main = (main é a pasta principal) envia todos os commits locais para a nuvem.
